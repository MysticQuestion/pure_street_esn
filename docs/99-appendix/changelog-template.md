## Changelog Template

The changelog records all notable changes for each release of ESN.  This template follows [Keep a Changelog](https://keepachangelog.com/) guidelines.

```
## [X.Y.Z] – YYYY-MM-DD
### Added
- A list of new features introduced in this release.

### Changed
- Updates to existing functionality, including backwards‑compatible changes.

### Deprecated
- Features that are still available but will be removed in future releases.

### Removed
- Features removed in this release.  State if there is a recommended alternative.

### Fixed
- Bug fixes.  Reference issue numbers if applicable.

### Security
- Security improvements or vulnerability fixes.

```

Use this template to add new sections for each release in `CHANGELOG.md`.  Always include the date of release and adhere to semantic versioning.  Keep the tone neutral and informative.
